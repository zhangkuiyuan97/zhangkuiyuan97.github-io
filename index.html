常见问题

这个网站如何部署到 GitHub Pages？
  1. 在 GitHub 新建仓库：your-username.github.io（用户主页）或任意名称（项目页）。
  2. 把本文件命名为 index.html 并提交推送到 main 分支根目录。
  3. 打开仓库 Settings → Pages，选择部署分支为 main / root。
  4. 几分钟后访问：https://your-username.github.io/ 或 https://your-username.github.io/仓库名/。
怎么自定义深色/浅色主题？

点击右上角“主题”，或在代码顶部的 :root / [data-theme] 中调整变量即可。
